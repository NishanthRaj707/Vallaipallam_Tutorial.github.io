Vallaipallam VNAS Command Documentation

A comprehensive guide to Vallaipallam's Network Administration System (VNAS) commands.

1. Zero-Argument Commands (0 Args)

These commands execute immediately and **do not** require any additional arguments.

Tamil Keyword Hindi Keyword English Command Functionality Description
நிர்வாகி_இயக்கு प्रशासक_रन RUN_AS_ADMIN Executes the next command with administrative/root privileges.
இணைப்பு_முகவரி इंटरफ़ेस_पते_दिखाओ SHOW_INTERFACE_ADDRESSES Displays the IP addresses and netmasks for all active network interfaces.
இணைப்பு_நிலை इंटरफ़ेस_स्थिति_දિਖාਓ SHOW_INTERFACE_STATUS Lists the operational status (up/down) of all configured network interfaces.
இணைப்பு_செயலில் सक्रिय_कनेक्शन_दिखाओ SHOW_ACTIVE_CONNECTIONS Shows a list of all currently established network connections (TCP/UDP).
இணைப்பு_உள்வெளி नेटवर्क_IO_दिखाओ SHOW_NETWORK_IO Displays the total network input/output (TX/RX) statistics since boot.
இணைப்பு_உள்வெளி_பிரிப்பு प्रति_इंटरफ़ेस_IO_दिखाओ SHOW_NETWORK_IO_PER_INTERFACE Displays network input/output (TX/RX) statistics broken down by interface.
இணைப்பு_வேகம் इंटरफ़ेस_गति_मापो SHOW_NETWORK_SPEED Tests and displays the current link speed and throughput of the primary interface.
வலைப்பின்னல்_பட்டியல் वाईफाई_सूची_दिखाओ LIST_WIFI Scans and lists all available Wi-Fi networks in the vicinity.
வாயில்_காட்டு गेटवे_दिखाओ SHOW_GATEWAY Displays the default network gateway IP address.
தடுப்பு_செயல்படுத்து फायरवॉल_सक्रिय_करो ENABLE_FIREWALL Activates the system's network firewall.
தடுப்பு_நிறுத்து फायरवॉल_निष्क्रिय_करो DISABLE_FIREWALL Deactivates the system's network firewall.
துறைமுகம்_விதிகள் फायरवॉल_नियम_दिखाओ LIST_FIREWALL_RULES Displays all currently configured firewall rules (allow/block).

2. One-Argument Commands (1 Arg)

These commands require exactly **ONE** argument to be specified.

Tamil Keyword Hindi Keyword English Command Argument Required Functionality Description
இணைப்பு_திற इंटरफ़ेस_सक्रिय_करो ENABLE_INTERFACE <interface_name> Brings the specified network interface (e.g., eth0) online.
இணைப்பு_மூடு इंटरफ़ेस_निष्क्रिय_करो DISABLE_INTERFACE <interface_name> Takes the specified network interface offline.
DHCP_அமை DHCP_सक्रिय_करो SET_DHCP <interface_name> Configures the specified interface to obtain an IP address via DHCP.
இணைப்பு_அடைந்ததா पिंग_जाँच PING <target_address> Sends ICMP echo requests to the target to test connectivity and measure latency.
வழித்தடம்_காட்டு मार्ग_दिखाओ TRACEROUTE <target_address> Displays the path and hop count that packets take to reach a destination.
பெயர்ப்பட்டியல்_காண் DNS_खोज DNS_LOOKUP <domain_name> Resolves the IP address associated with the given domain name.
தாமதம்_சோதனை विलंब_परीक्षण LATENCY_TEST <target_address> Measures the network latency (delay) to the target address.
துறைமுகம்_அனுமதி पोर्ट_अनुमति ALLOW_PORT <port_number> Adds a firewall rule to allow incoming traffic on the specified port.
துறைமுகம்_தடு पोर्ट_अवरोध BLOCK_PORT <port_number> Adds a firewall rule to block incoming traffic on the specified port.
சேவை_அனுமதி सेवा_अनुमति ALLOW_SERVICE <service_name> Allows traffic for a predefined service (e.g., HTTP, SSH) through the firewall.
சேவை_தடு सेवा_अवरोध BLOCK_SERVICE <service_name> Blocks traffic for a predefined service through the firewall.
வலைப்பின்னல்_துண்டிக்க वाईफाई_काटो DISCONNECT_WIFI <network_name> Disconnects the system from the specified Wi-Fi network.

3. Multi-Argument Commands

3.1. Two-Argument Commands (2 Args)

These commands require **TWO** main arguments, separated by a keyword argument.

Tamil Keyword Hindi Keyword English Command Arguments Required (Syntax) Functionality Description
வலைப்பின்னல்_இணை वाईफाई_जोड़ो CONNECT_WIFI <network_name> **PASSWORD** <password> Connects to a password-protected Wi-Fi network using the given credentials.
துறைமுகம்_சோதனை पोर्ट_जांच PORT_SCAN <target_address> **PORT** <port_range> Scans the target address for open ports within the specified range (e.g., 1-1024).

3.2. Three-Argument Commands (3 Args)

This command requires **THREE** main arguments, separated by two keyword arguments.

Tamil Keyword Hindi Keyword English Command Arguments Required (Syntax) Functionality Description
இணைப்பு_IP_அமை इंटरफ़ेस_IP_सेट_करो SET_IP <interface_name> <ip_address> **GATEWAY** <gateway> **NETMASK** <subnet_mask> Statically assigns an IP address, gateway, and subnet mask to the specified interface.
